### Proyecto Principal

#### [Portafolio Interactivo 3D](https://portafolio-elias-sanchez-production.up.railway.app)
Un espacio digital interactivo diseñado para demostrar las capacidades del navegador moderno:
* **Fondo 3D Nebuloso:** Miles de partículas calculadas por GPU e interpoladas matemáticamente en tiempo real al hacer scroll.
* **Micro-interacciones avanzadas:** Un cursor con física elástica de gel y efectos interactivos de decodificación y descifrado de texto.
* **Paleta de Comandos Glassmorphic:** Una interfaz minimalista de consola (`Ctrl + K`) para navegación rápida y control del tema visual del sitio.
